# Python Fundamentals – Bootcamp Project

This repository contains exercises and small programs completed as part of my **Data Technician bootcamp (Week 6 – Python)**. The goal of the project was to build a solid foundation in Python by solving practical programming tasks and creating simple interactive programs.

## What I Practiced

Throughout the exercises I worked with core Python concepts, including:

- **Variables** to store and manipulate data
- **`print()`** to display results and feedback to users
- **`input()`** to collect user input and make programs interactive
- **Type casting** (e.g. `int()`, `float()`, `str()`) to convert between data types
- **Conditional logic** using `if`, `elif`, and `else`
- **Loops** including `for` and `while` to repeat operations and control program flow

These elements were combined to create program logic, perform calculations, and interact with users.

## Mini Projects

### FizzBuzz
Implemented the classic programming challenge using loops and conditional logic:

- Iterate through numbers **1–100**
- Print:
  - `"fizz"` if divisible by 3
  - `"buzz"` if divisible by 5
  - `"fizzbuzz"` if divisible by both
  - Otherwise print the number itself

This exercise helped reinforce **control flow and logical conditions**.

### ATM Simulation
Built a simple command-line ATM program that allows a user to:

- View account balance
- Deposit money
- Withdraw money
- Exit the system

The program uses:

- **Functions** to organise logic
- **Conditional statements** to control menu options
- **Loops** to keep the program running until the user exits
- **User input validation** for withdrawal amounts

This project tied together multiple Python concepts into a small but realistic interactive application.
